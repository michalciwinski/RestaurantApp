# RestaurantApp
Application as an extension for mobile waiter robot with implemented OpenAI Assistant

# Technologies
BACKEND:
- C#/.NET
- Entity Framework 
- PostgreSQL

FRONTEND:
- REACT.JS

ADDITIONAL:
- OpenAI API
